# kura-ui

A UI component library and NUI runtime for FiveM (FXServer). Built as the interface layer for [Project Kura](https://kura.walteria.net).

---

## Requirements

- FXServer (FiveM)

---

## Installation

> **Do not use "Code -> Download ZIP."** Download the prebuilt release instead.

1. Download the latest release from the [Releases](../../releases) page.
2. Extract into your `resources` folder.
3. Add to your `server.cfg`:

```cfg
ensure kura-ui
```

---

## Building from Source

Requires [Bun](https://bun.sh).

```sh
bun install
bun run build
```
